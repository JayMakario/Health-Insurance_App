 ## Health Insurance Cost Prediction App

 ## Project Overview
This project predicts health insurance costs using machine learning to support better risk assessment and pricing decisions in insurance systems.

 ## Business Problem
Insurance companies need accurate cost predictions to:
- Reduce underwriting risk
- Improve pricing strategies
- Enhance customer segmentation

  ## Approach
- Exploratory Data Analysis
- Data preprocessing & feature engineering
- Model training (Linear Regression, Random Forest, SVR, Polynomial Regression, XGBoost)
- Model evaluation using RMSE, MAE and R²
- Deployment using Streamlit

 ## Results
- Best Model: XGBoost
- RMSE: 5008.93
- R² Score: 0.83		
- MAE: 3864.69
